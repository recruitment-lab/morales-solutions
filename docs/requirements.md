
# Minimum Requirements:

- Navbar with Logo, fixed (visible after scrolling)
- Hero/Header
- Footer
- After the footer, a Copyright section linking to this page, with © 2026 All rights reserved. (display current year with Javascript)
- Screenshots are available in the folder: 'public/screenshots/'
- You can freely use the icons in 'public/icons'
- Use itshover.com icons like https://www.itshover.com/icons/airplane-icon, which were installed using "npx shadcn@latest add https://itshover.com/r/plug-connected-icon.json"
